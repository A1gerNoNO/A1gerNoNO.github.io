这是这个网站的第一篇文章。

本站使用 [Hugo](https://gohugo.io/) 生成静态页面，使用 Ivy 主题，并通过 GitHub Actions 部署到 GitHub Pages。

## 为什么选择 Hugo

Hugo 的优点是构建速度快、结构清晰、部署简单。对于个人博客、学术主页、项目文档和学习笔记，它都很合适。

## 为什么选择 Ivy

Ivy 是一个极简双栏主题，适合专注于文字内容。它不会把页面做得很复杂，比较适合长期写作。
