这是这个网站的第一篇文章。

本站使用 [Hugo](https://gohugo.io/) 生成静态页面，使用 PaperMod 主题，并通过 GitHub Actions 部署到 GitHub Pages。

## 为什么选择 Hugo

Hugo 的优点是构建速度快、结构清晰、部署简单。对于个人博客、学术主页、项目文档和学习笔记，它都很合适。

## 为什么选择 PaperMod

PaperMod 是一个干净、响应式、适合技术博客的 Hugo 主题。它自带明暗模式、文章列表、代码复制按钮和较好的移动端体验。
